# 目薬通知bot
目薬通知botは特定の時間に通知を送るためのbotです．機能としてはpush通知の他にメッセージのやり取りも行うことができます．

# なぜ作成したのか
母がドライアイで毎日定期的に目薬を指す必要があるが，よく「忘れてた」と言っていました．そこで，特定の時間に母に通知を送ることができればこの問題を解決することができるのではないかと考え，この通知botを作成しました．

# 使用技術
### バックエンド
Python 3.9.1
Django 3.2.7
### API
LINE Messaging API
### 本番環境
Heroku
